## 🌟 Features

- 🎨 **Vibrant, colorful UI** inspired by Asian Paints & premium interior brands
- ✨ **Framer Motion animations** — paint splash, hover effects, floating cards, scroll reveals
- 🏠 **Complete service showcase** — Modular Kitchens, Wardrobes, False Ceiling, PVC Panels, Paint Solutions & more
- 🖼️ **Interactive Paint Experience** — click colors to see live room preview + liquid splash animation
- 🗂️ **Masonry Gallery** with category filtering and hover zoom
- 📱 **Fully responsive** — mobile-first premium experience
- 🏷️ **Trusted Brands** — Asian Paints, Berger, Indigo, Birla, Ashirvad, J.K. Putty, Sleek Kitchens
- 📞 **Contact section** with direct call buttons and consultation form

## 🚀 Getting Started

